# ASPIRE | The Agency

Ready-to-upload Next.js + Tailwind project.

## Install
```bash
npm install
```

## Run
```bash
npm run dev
```

## Build
```bash
npm run build
npm start
```

## Notes
- Premium event and conference agency landing page.
- Mobile-first and compressed for less scrolling.
- Uses stock imagery for a stronger visual result.
- Uses only the business information provided.
